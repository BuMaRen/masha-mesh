## 1. 基线核对（当前状态）

### 已具备（核对后勾选）
- [x] 启动层已将共享 cache 注入 webhook 和 reconciler
- [x] webhook 已具备最小注入判定（label + 同名容器检查 + cache 命中 patch）
- [x] controller 已区分 Add/Update/Delete 并处理 Deployment/StatefulSet
- [x] 已有少量单测覆盖 webhook 纯函数与容器移除逻辑

### 已知缺口（修复后勾选）
- [x] cache 主键缺少 namespace 维度
- [x] 注入契约隐式，label key/value 缺少明确语义
- [x] webhook 幂等仅“同名存在即跳过”，不可证明版本一致。`版本不一致的话交由controller处理`
- [x] cache miss 失败语义未参数化（当前等价 fail-close）。`预期就是 fail-close`
- [ ] Admission 输入校验不完整（Kind/Operation/SubResource）
- [ ] reconciler 目标筛选过宽（仅按 label key）
- [ ] reconciler Plan/Apply 未拆分
- [ ] workload 更新无冲突重试、退避、失败分类
- [ ] 删除路径缺少“容器列表非空”等安全护栏
- [ ] 注入链路可观测性不足（success/fail/conflict/cache miss/latency）
- [ ] 关键行为测试不足（mutate handler、收敛、冲突、部分失败）
